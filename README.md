## Description

This is a minimalist React validation using Typescript, Material UI, Redux, React.

## How to run

### Local development

- Clone the project and cd into project
- npm install
- npm start and go to [link](*http://localhost:3000*)

### Deployment

- npm install
- npm run build
- npm run deploy

## Key technologies & Libraries used

- Material UI (1.1.0)
- Redux
- Typescript
- React
